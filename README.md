# V Rising Macro Manager (Bloodcraft UI)

AutoHotkey v2 macro manager for V Rising chat commands.

## Features

- GUI to add/update/delete hotkeys
- Actions: SendChat, PreSendChat, StashNGrab, GrabBuffs
- Bloodcraft / Light theme toggle
- INI persistence
- Duplicate hotkey warnings + highlight

## Requirements

- AutoHotkey v2 https://www.autohotkey.com/v2/

## Run

Double-click/Single-click `V_Rising_Macro_Manager.ahk`

## Notes

- Default config is stored in `vrising_macros.ini` (ignored by git).
